# Telemetry Client

This directory contains the client applications for the Operating System Analysis project.

## Structure

- **`linux/`**: Contains the Linux shell script client (`client.sh`).
- **`windows/`**: Contains the Windows PowerShell client (`client.ps1`).

## Usage

### Linux
Run the shell script:
```bash
cd linux
./client.sh
```

### Windows
Run the PowerShell script:
```powershell
cd windows
.\client.ps1
```
